Pageflip5 Features

*constant 60fps animations are available when all content is loaded, on today's middle range desktop computers, and latest ipads, iphones.

Mobile safari & mobile chrome are supported

When you flip a page, you don’t need to wait until animation is finished, just click on the corner or press left/right key to flip another page, and another...

Flipping animation isn’t skipped when a new one is started.

see the 60fps pageflip5 feature video

Pageflip5 supports all flipping directions you need, for all languages, or content type

You can place more transparent pages one after another, and will see through all of them, just like as it is in this book

Many configuration options are available to fine tune pageflip5 behaviors, appearance.

all the power of CSS3 and HTML5 to make amazing books, with live effects, animations.

 

Pageflip5 is lightweight. To run it, you only need jQuery and a small 44KB javascript library.

Only needed pages are loaded into memory, this let you make very large books without any side effects, like animation lags, and browser memory overflow.

You also have the option to not unload some pages which needs to be in memory.

You have many options to select how pageflip5 displays your book.

By default the book is scaled to fit available space. IF your book is scaled down to fit the stage, the zoom feature become available to view your book in its original resolution.

You can choose which page have thumbnails, and use them like a table of contents

you aren’t forced to use page numbering
from 1 to the last page number:

you can set custom page numbers, region start numbering or even names to display it.

You can also add labels to the pages, for easy internal usage, for accessing page. It’s useful if you add more pages later, while having links from one page to another.

Each page have its own URL (using hash tags) and you can link to it from another pages, or from your website.

If you’re browsing a pageflip5 book, and want to share a content on a page, just copy the URL from the browsers addressbar, and it will point to that page.

Demo you book using autoflip, your audience have just to sit back and watch your content like a slideshow.

Auto flipping interval can be set globally or even at page level, if some pages need more time to be visible.
